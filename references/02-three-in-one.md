# 板块四详解：单条视频「三位一体」深度复盘

**这是 v3 与旧版最大的区别，绝不只是摆数据。**

每条视频严格按以下结构输出（可先给一张精简数据表，再逐条三位一体解读）：

```
[视频标题] · [发布时间] · [类型标签：学员采访/口播观点/直播切片]
① 数据反馈结果：播放 X（周均的 N 倍/几分之一）、点赞 X、评论 X、分享 X、涨粉 X
   —— 先回答"这条最终跑出了什么、相对大盘是高还是低"
② 过程指标诊断：完播 X%（高于/低于均值）、2s 跳出 X%、平均时长 Xs、封面 CTR X%
   —— 回答"为什么是这个结局"（留人能力如何）
③ 内容拆解：选题切中___；开头钩子___；结构（场景→判断→行动？）___；标题___；封面___
   —— 回答"内容层面做对了/做错了什么"
─────────────────────────────────────────
④ 为什么好 / 为什么不好：（把①②③串成一条因果链，不是各说各话）
⑤ 可复用：哪些方法能原样复制到其他选题（如"本科一般的孩子"破题法）
⑥ 可沉淀：哪些能变成内容资产（观点模板/封面模板/标题模板/可复用的高完播结构）
⑦ 是否闭环：内容→数据反馈→优化→再内容，形成飞轮了吗？
   （如：直播切片→评论区引导具体问题→下场直播选题，就是闭环）
```

**红线**：每条都必须有 ④⑤⑥⑦ 的深层复盘，缺一不可；表现严重翻车的，⑥ 用「必须修改提升」红色标题。

---

## 直播切片重点拆解

若某条切片播放破万，单独加"深度拆解"小节，用上面 ①~⑦ 全量展开，并特别强调**不是数据罗列**，而是在数据基础上做更深层的因果与闭环分析：

- **其高完播/低跳出的根因**：开头钩子如何一秒击中人群、中段如何用"判断+证据"留人。
- **可复制的结构**：把它抽象成可套用到其他选题的母版（如「具体身份 + 强决策题 + 场景→判断→行动」）。
- **对直播间的长效导流价值**：是否在多场直播持续出现在 TOP 引流位、曝光/进房/转化如何，验证"切片→直播→评论区引导→下场选题"飞轮是否已转起来。
- **下一步动作**：把这个偶发爆款变成每周固定动作（固定流水线 / 模板化）。

> 注意：封面上的「直播切片」角标是图片文字，OCR 未必能直接识别（见 `references/06-ocr-guide.md`）。若无法从截图确认，可依据"播放量破万 + 与直播引流强关联"做代表性归类，并在报告中注明"以用户目测封面为准"，请用户最终核对。
